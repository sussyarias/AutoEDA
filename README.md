# AutoEDA
AutoEDA is a Python application built with Streamlit that helps automate Exploratory Data Analysis (EDA).

The goal of this project is to simplify the initial data analysis process by allowing users to upload CSV or Excel datasets and quickly explore their structure, statistics, and patterns.

## Current Features
* Upload CSV and Excel files
* Automatic detection of variable types
* Basic data cleaning
* Descriptive statistics
* Interactive visualizations

## Technologies
* Python
* Streamlit
* Pandas
* Plotly

## Status
This project is currently under development and new features are being added progressively :)
